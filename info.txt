
Самый быстрый — через Docker:
docker run -p 8080:8080 flowable/flowable-ui

После этого открой браузер:
http://localhost:8080/flowable-ui

Внутри будет Modeler, Task app, Admin, IDM.
Можно сразу рисовать BPMN, сохранять и запускать процессы через REST.

Сonstraint:
- PRIMARY KEY                            - FOREIGN KEY
CREATE TABLE users (                     CREATE TABLE orders (
    id INT PRIMARY KEY,                      id INT PRIMARY KEY,
    name VARCHAR(100)                        user_id INT,
);                                           FOREIGN KEY (user_id) REFERENCES users(id)
                                         );
- UNIQUE                                 - NOT NULL
CREATE TABLE users (                     CREATE TABLE users (
    id INT PRIMARY KEY,                      id INT PRIMARY KEY,
    email VARCHAR(100) UNIQUE                name VARCHAR(100) NOT NULL
);                                       );
- CHECK                                  - DEFAULT
CREATE TABLE users (                     CREATE TABLE users (
    id INT PRIMARY KEY,                      id INT PRIMARY KEY,
    age INT CHECK (age >= 18)                role VARCHAR(50) DEFAULT 'USER'
);                                       );
- INDEX (не совсем constraint)
CREATE INDEX idx_email ON users(email);
